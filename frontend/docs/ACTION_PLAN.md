## 🆘 If Something Goes Wrong

### Problem: Download button not visible
**Solution**: Use direct link:
```
https://nodejs.org/dist/v24.14.0/node-v24.14.0-x64.msi
```

### Problem: Installation fails
**Solution**:
1. Uninstall Node.js (Settings → Apps → Apps & features)
2. Restart computer
3. Download again
4. Run installer as Administrator

### Problem: npm still not recognized after restart
**Solution**:
1. Restart computer again
2. Open new PowerShell window
3. Try `npm --version` again

### Problem: Port 3000 already in use
**Solution**:
```bash
set PORT=3001 && npm start
```
Then open: http://localhost:3001
